# Сложно сосредоточиться

Учебный проект по адаптивной вёрстке из курса **Яндекс Практикум**. 
Это лендинг-статья о проблемах с концентрацией внимания и способах борьбы с ними. Главная техническая особенность проекта — реализация светлой и тёмной тем оформления с возможностью автоматического и ручного переключения.

Логика работы переключателя уже была реализована в готовом JavaScript-файле, я работал над HTML-структурой и написанием продвинутых CSS-стилей.

## Что умеет страница

- Адаптируется под любые размеры экранов (от мобильных телефонов до широких десктопных мониторов).
- Поддерживает три визуальные темы: светлую («день»), тёмную («неон») и автоматическую («авто»).
- Умеет автоматически подстраиваться под системные настройки устройства пользователя (светлая или тёмная тема в ОС).
- Плавно масштабирует шрифты заголовков в зависимости от ширины экрана.

## Технологии

- HTML5, строгая семантическая разметка.
- CSS3 (широкое использование CSS-переменных, Grid Layout, Flexbox).
- Подход Mobile First к написанию стилей.
- Адаптивная типографика с использованием математической функции `clamp()`.
- Использование логических свойств CSS (например, `margin-block-start` вместо `margin-top`).
- Работа с медиафункцией `prefers-color-scheme`.
- Подключение и использование кастомного шрифта IBM Plex Mono.
